# Codex 문서 지도

`mydocs/manual/codex/`는 Codex가 rhwp 저장소에서 재현 가능한 작업 절차를 찾기 위한 진입점이다.
현재 작업 규칙의 권위 문서는 [문서·Git 워크플로](docs_and_git_workflow.md)이며, PR 처리와 검증은 그
문서가 연결하는 각 canonical manual을 따른다.

## 현행 문서

- [문서·Git 워크플로](docs_and_git_workflow.md): 문서 위치, 브랜치, 승인, 커밋의 공통 절차

## 과거 메모리

- [archive](archive/): 2026년 5월 특정 세션, 종료 task, 당시 환경을 기록한 historical snapshot

이전 `MEMORY.md` 경로는 외부 이력 호환용 redirect로만 보존하며 활성 문서의 진입점으로 링크하지 않는다.
`archive/` 문서는 현재 경로, 브랜치, 도구 동작 또는 작업 절차의 근거로 사용하지 않는다. 필요한 장기
교훈은 현행 canonical 문서에 다시 검증해 반영한다.
