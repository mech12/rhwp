# Task #1755 수행계획서 — 지연 자리차지 표 host 제목 줄의 이월-전-쪽 잔류

## 배경
#1753(후속 텍스트 선행 채움)의 한계 항목 후속. 대표 2814765: 한글은 host 제목 줄
("1) 투입인원수 산정기준")을 **9쪽 하단**(이월 전 쪽)에 렌더하는데, rhwp 는 layout 의
`defer_visible_rowbreak_host_text` 경로로 **마지막 fragment 뒤(11쪽)** 렌더 — 제목이 자기
표와 후속 문단들(#1753 후 9쪽) 뒤에 나타나는 순서 결함(SVG 확인).

## 수정 방향
hidden_empty_paras 와 동일한 typeset→layout 신호 전달 패턴:
1. typeset: #1753 prefill 가드 성립 시(이월 직전), host 텍스트 줄을
   `PartialParagraph{0..line_count}` 로 현재 쪽에 pre-emit + 줄 높이 소비,
   `TypesetState.pre_emitted_host_paras` 기록. prefill 은 그 뒤 수행(한글 순서:
   제목 → 후속 문단).
2. `PaginationResult.pre_emitted_host_paras` 신설(typeset 결과 복사; Paginator 는 빈 집합).
3. layout: `set_pre_emitted_host_paras()` 세터 + `render_deferred_rowbreak_host_text_after`
   에서 pre-emit 문단 억제(이중 렌더 방지). 호출부(rendering.rs)에
   set_hidden_empty_paras 와 병렬 배선.
