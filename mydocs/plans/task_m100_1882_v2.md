## 1. 배경

PR #1890(C1c) merge 시 메인테이너 확인 사항: 한컴 `2차원원형` 정답지의 제목은 **"판매"**인데
rhwp는 자동 placeholder **"차트 제목"**으로 렌더. XML `c:title`에 명시 텍스트가 없는 것은
동일하므로, 한컴의 자동 제목 정책에 시리즈 이름 fallback이 있는지 확인 요청됨
(`mydocs/pr/archives/pr_1890_review.md`, #1882 코멘트).

## 3. 목표 / 범위

- `effective_title` 우선순위를 **명시 제목 → (단일 시리즈) 시리즈 이름 → "차트 제목"**으로 확장.
  게이트(`has_title_elem && !auto_title_deleted`)는 기존 유지.
- 수정 파일: `src/ooxml_chart/renderer.rs` 단독 + 통합 테스트 1파일 보강. 모델·파서 무변경.
