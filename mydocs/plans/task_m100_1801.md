# Task #1801 수행계획서 — visual sweep 일반 파일 입력 및 overlay 검토 개선

## 배경

`scripts/task1274_visual_sweep.py`는 기존 preset target 중심이라 PR 리뷰나 시각 검증 중 임의
HWP/HWPX 파일과 기준 PDF를 바로 비교하기 어렵다. 또한 특정 페이지만 빠르게 확인하거나, compare와
overlay를 한 화면에서 보며 자동 일치율 보조값을 함께 확인하는 흐름이 부족했다.

## 목표

- preset target 외 일반 파일 경로 입력을 지원한다.
- 특정 페이지 또는 페이지 범위만 비교할 수 있게 한다.
- rhwp/PDF PNG overlay diff와 페이지별 metric을 생성한다.
- compare와 overlay를 나란히 붙인 review PNG를 생성한다.
- review PNG의 overlay 바로 아래에 내용 픽셀 중심 자동 일치율 보조값을 한 줄로 표시한다.
- `mydocs/manual/verification/visual_sweep_guide.md`에 Codex 보고 규칙과 해석 기준을 남긴다.

## 구현 범위

- `scripts/task1274_visual_sweep.py`
  - `--hwp`, `--pdf`, `--key`, `--file-target` 추가
  - `--page`, `--pages` 추가
  - `overlay/overlay_###.png`, `overlay_metrics.json` 추가
  - `review/review_###.png`, `review_contact_sheet.png` 추가
- `mydocs/manual/verification/visual_sweep_guide.md`
  - 일반 파일/특정 페이지 사용법
  - Codex 화면 표시와 보고 형식
  - `visual_accuracy_proxy_percent` 해석
