## 배경

문서 PR #1701은 #1702 merge 전 정책/측정 원천 문서를 추가했고, 코드 PR #1702는 `Build & Test` cargo
cache save를 trusted branch인 `devel` / `main` push로 제한했다.

#1702 merge 이후에는 다음 후속 관측이 GitHub 이슈 코멘트에 먼저 기록됐다.

- #1664 상세 보고: <https://github.com/edwardkim/rhwp/issues/1664#issuecomment-4853263686>
- #1668 부모 롤업: <https://github.com/edwardkim/rhwp/issues/1668#issuecomment-4853268643>

메인테이너 요구사항인 `mydocs/`: 정책 / 의사결정 / measurement 기록을 맞추기 위해, 이 후속 관측값을
이슈 코멘트에만 두지 않고 원천 문서에 보존한다.
