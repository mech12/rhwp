# 수행계획서 — Task M100 #2228: Canvas2D 정적 그림 레이어의 표 셀 clip 보존

- 재현 파일: `samples/issue2217/20200830.hwp` 2쪽
- 기준 자료: `pdf/issue2217/20200830-2020.pdf` 2쪽

## 문제

기본 Canvas2D 경로는 정적 flow 그림을 DOM layer로 분리한다. 이때
`PageLayerTree`의 `clipRect(tableCell)` 문맥을 버리고 그림 bbox만 수집해, 표 셀 밖에
있어 한컴에서 잘려야 하는 그림까지 화면에 표시한다. 이 문서의 로고 표 셀 [0,2]에서는
붉은 `창립교회` 그림이 반복되어 보인다.

## 구현 방향

1. `collectFlowImagePaintOps()`가 트리를 순회할 때 상위 `clipRect`를 누적하고 교집합을
   각 DOM flow image에 전달한다.
2. DOM image layer는 누적 clip이 있는 그림을 clip wrapper 안에 배치한다. clip 밖에 완전히
   있는 그림은 생성하지 않는다.
3. clip이 없는 일반 flow image의 DOM 분리, crop, 회전/대칭 및 기존 재렌더 정책은 유지한다.
4. 샘플명이나 페이지 번호가 아닌 `PageLayerTree`의 구조적 clip 정보만 사용한다.

## 검증 계획

1. TypeScript 단위 테스트로 중첩 clip 교집합과 clip 없는 그림의 기존 동작을 검증한다.
2. `npm` Studio 테스트와 build를 수행한다.
3. `wasm-pack build --target web --out-dir pkg` 후 `20200830.hwp` 2쪽을 기본 Canvas2D와
   CanvasKit에서 확인한다.
4. `#2226` 표 행높이 회귀 테스트를 함께 실행해 주소 블록/행높이가 유지되는지 확인한다.
