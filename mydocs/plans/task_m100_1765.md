# Task #1765 수행계획서 — 병합 셀 경로 trailing ls 가설 검증 (#1763 후속)

## 배경
#1763 은 rs=1 셀 경로의 선언높이 권위 가드만 적용했다. 후속 가설은 병합 셀 경로
(height_measurer 2-b)도 동일한 trailing ls 포함 측정으로 required 가 부풀어 마지막 스팬 행을
deficit 확장한다는 것이었다. 재현 후보는 17931383 표의 #1759 하니스 dh **+5.2px**
(#1763 후 잔존)이다.

## 검증 방향
1. 후보 표의 셀 rowspan, 행별 선언높이, 렌더 행높이를 대조한다.
2. 초과분이 실제 병합 셀 경로에서 발생했는지 확인한다.
3. 초과분이 전적으로 마지막 줄 trailing ls 로 설명되는 경우에만 2-b 경로 가드를 검토한다.

## 판별 결과
Stage 2에서 가설을 기각했다.

- 17931383 후보 표의 셀은 전부 `rs=1` 이며 병합 셀 경로를 지나지 않는다.
- +5.2px 는 row9 단일 행의 per-line 콘텐츠 측정 누적으로 재분류했다.
- 따라서 소스 수정 없이 조사 문서와 샘플만 남긴다.

## 회귀 게이트
1. HWP/HWPX 샘플과 한컴 2024 기준 PDF의 페이지 수 및 문제 페이지 구조를 확인한다.
2. 샘플은 #1759 per-line 콘텐츠 측정 누적의 대표 재현 사례로 동결한다.
