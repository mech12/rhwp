# Task M100 #1634 stage5 — 부분 선택 표 내부 전치 붙여넣기

## 문제

stage4 이후 전체 표 선택은 기존 표 자체 전치로 처리된다. 그러나 부분 선택은 무조건 새 표 생성 경로로 분기한다.
따라서 전치 복사 버퍼를 기존 표 안의 선택 범위에 붙여넣을 수 없다.

## 목표

- 셀 선택 모드에서 선택 범위가 표 전체이면 기존처럼 표 자체 전치를 수행한다.
- 선택 범위가 표 일부이면 선택 범위의 좌상단을 대상 시작 셀로 삼아 기존 표 안에 전치 붙여넣기를 먼저 시도한다.
- 대상 범위가 표 크기를 초과하는 경우에만 새 표 생성 경로로 폴백한다.
