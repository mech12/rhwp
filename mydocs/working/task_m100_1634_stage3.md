# Task M100 #1634 stage3 — 셀 선택 상태 전치 붙여넣기 분기

## 문제

셀 범위를 선택한 상태에서 컨텍스트 메뉴의 `셀 전치 붙여넣기`를 실행하면 현재 커서가 여전히 원본 표 셀 안에
있어 기존 표 내부 붙여넣기 분기를 탄다. 원본 표가 `4×2`이고 전치 결과가 `2×4`인 경우 대상 범위가 원본 표의
열 수를 초과하여 아래 오류로 실패한다.

```text
셀 범위 (0,0)~(1,3)가 표 크기 4×2를 초과합니다
```

## 목표

- 셀 선택 모드에서 `셀 전치 붙여넣기`를 누르면 원본 표 내부 덮어쓰기가 아니라 새 전치 결과 표 생성 경로를 탄다.
- 표 내부의 일반 커서 위치에서 붙여넣는 기존 동작은 유지한다.
- 기존 표 내부 붙여넣기가 대상 범위 초과로 실패할 때도 새 전치 결과 표 생성으로 폴백한다.
