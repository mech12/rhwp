## 판별 (수행계획서 가설 대비)
- (a) 상수 계열 = **측정 노이즈 확정** (테두리 아티팩트 가설보다 단순 — 행별 반올림 누적).
  실 결함 아님. 하니스 해석 시 |dh| < 1px 는 노이즈로 간주해야 함.
- (b) 행비례 계열 = **기각**. 실체는 특정 행(대형 폰트 제목/복합 콘텐츠)의 콘텐츠 높이 차 —
