## 분석 내용

- `./target/debug/rhwp ir-diff samples/SO-SUEOP.hwpx samples/SO-SUEOP.hwp -s 0 -p 574 --max-lines 120`로 p22 관계도 문단을 비교했다.
- HWP3 p22 관계도 내부 텍스트는 HWPX와 원문 IR 자체가 다르다.
  - HWPX: 관계도 셀 내부에 circled number와 연결선 PUA가 포함된다.
  - HWP3: 파서 출력 기준 셀 텍스트가 `윤직원 윤창식 윤종수 윤경손 | 춘섬... 윤종학`처럼 단순 텍스트로 남는다.
- Stage 7의 `obj_type=1 -> Table IR` 보정은 관계도 상자의 외곽선과 `TopAndBottom` 자리차지 흐름을 복원한다.
- 그러나 관계도 내부 선/번호 완전 복원은 글상자/도식 내부 표현을 추가로 해석해야 하므로, #1692의 글색상 손실 및 페이지 흐름 보정 범위를 넘는다.

## 판단

- p22/p6 등 도식 내부 선/번호 표현은 별도 HWP3 도식 파싱 후속 이슈로 분리한다.
