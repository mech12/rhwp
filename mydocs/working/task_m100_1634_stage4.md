# Task M100 #1634 stage4 — 기존 표 자체 전치

## 문제

stage2/3는 셀 선택 상태의 전치 붙여넣기 실패를 새 표 생성으로 우회했다. 그러나 실제 기대 동작은 선택된 기존 표가
`4×2 → 2×4`처럼 구조 자체를 바꾸는 것이다.

## 목표

- 전치 복사 후 셀 선택 상태를 유지한다.
- 선택 범위가 표 전체이면 `셀 전치 붙여넣기`가 기존 표 자체를 전치한다.
- 원본 표가 별도의 새 표로 복제되지 않는다.
- 표 전체가 아닌 부분 선택은 기존 새 표 생성 경로를 유지한다.

## 범위

- 1차 in-place 전치는 병합 없는 전체 표 선택만 지원한다.
- 병합 셀, 중첩 표 경로는 기존처럼 거부하거나 새 표 생성 경로로 남긴다.
- 열 폭은 기존 표 전체 폭을 유지하며 새 열 수에 맞춰 균등 분배한다.
