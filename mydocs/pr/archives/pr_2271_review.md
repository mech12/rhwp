## PR 본문 요약

PR은 [PR #2257](https://github.com/edwardkim/rhwp/pull/2257) 병합 직후 발견한, 같은 문단에 앵커된 `TopAndBottom` 그림 두 개의 pushdown 높이 이중 예약을 다룬다.

본문의 설명에 따르면 `17809123` 샘플에서 좌우로 나란히 배치된 그림을 기존 회계가 각각 합산해 trailing 빈 문단을 2쪽으로 밀었다. 수정안은 가로로 겹치는 float은 세로 스택으로 합산하고, 가로로 분리된 float은 같은 세로 band를 공유하는 것으로 보아 컬럼별 최대 높이만 예약한다. 단일 float의 기존 동작은 유지한다고 설명한다.

본문은 중간 세로-offset union 접근이 `1342000` 샘플을 75쪽에서 74쪽으로 회귀시킨다고 보고, 가로-컬럼 모델을 선택한 근거로 제시한다. 대표 샘플의 2쪽에서 1쪽 회복과 추가 문서 6건의 페이지 수 개선도 함께 주장한다.
