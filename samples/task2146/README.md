- 출처: 자치법규정보시스템 공개 문서 — 구미시 지방공무원 인사 규칙
  [별표 7] 전직시험이 면제되는 자격증 구분표 (개정 2021.6.10.)
- 구조: 78행×5열 RowBreak 표 (사선 헤더 셀 포함), 본문 비고 문단 pi=5..9
- 한글 2022: 6쪽. 수정 전 rhwp: 7쪽 (pi=5..9 가 7쪽으로 밀림).

### 결함 재현 (수정 전)

r0 사선 헤더 셀("계급|직류", "|직렬")은 저장 LINE_SEG 가 전혀 없는(NO_LS)
2문단 셀이다. composer 재합성 줄높이가 문단당 37.76px 로 부풀어 행높이
79.3px — 한글 실측 52.4px(= 선언 셀높이 3928HU 그대로) 대비 +26.9px.
이 팽창이 표 꼬리를 밀어 비고 문단(pi=6..8)이 7쪽으로 넘어갔다.

### 수정 (#2146)

전 문단 NO_LS + 모두 1줄(폭 여유) + 선언 내부높이 ≥ em 합인 셀 중 아래 두
경우 선언 셀높이 신뢰 (#1763/#2097 선언 권위 계열,
`composer::no_ls_short_label_cell`):

- (a) **사선 셀** (r0 c0 "|직렬", BF=13) — 한글은 사선 셀 문단을 일반 텍스트
  흐름으로 배치하지 않고 코너 라벨로 그린다.
- (b) **고정(Fixed) 줄간격 모순 셀** (r0 c1 "계급|직류": ps Fixed 37.76px
  ×2문단 > 선언 내부 48.6px) — 저장 스타일과 저장 지오메트리 충돌 시 한글은
  지오메트리(선언)를 유지한다.

사선 없는 일반 라벨 셀은 제외 — 한글이 fresh 레이아웃으로 선언 이상 키우는
문서(#1891 76076 규제영향분석서 계열)가 존재한다.

잔여: r0 수정(-26.9px) 후에도 이 문서는 잔여 팽창 +473px(24개 행, 이질
원인)로 rhwp 7쪽(한글 6쪽) 유지 — 후속 백로그.
